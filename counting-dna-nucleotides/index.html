ROSALIND｜Counting DNA Nucleotides (DNA)

發表於 2024-05-01 分類於 bioinformatics 閱讀次數： 閱讀次數： Disqus： 文章字數： 628 所需閱讀時間 ≈ 2 分鐘

Given: A DNA string s of length at most 1000 nt.

Return: Four integers (separated by spaces) counting the respective number of times that the symbols ‘A’, ‘C’, ‘G’, and ‘T’ occur in s.
（給定 DNA 字串，回傳四鹼基符號於字串內的數量）

Solution

python

在此例使用 dictionary 的 key 記錄鹼基符號，利用 value 記錄出現頻率，透過迴圈遍歷整個字串，每逢與 key 相同的字符即增加該 key 之 value。

儘管統計鹼基的方法很直觀，但輸出的順序要留意。題目要求以 A、C、G、T 之順序輸出鹼基出現頻率，但 python 的 dictionary 為無排序的雜湊表，所以值不會依照字母的順序或定義時的順序印出。為了回傳排序後的統計量，可以手動調整輸出順序，或是透過 OrderedDict 物件取代原生的 dictionary 儲存統計量。

OrderedDict 為 dictionary 的子類，OrderedDict 的 key 依插入順序排序，但可配合 sorted() 依 key 或 value 排序。具體的做法是使用 items() 取出 key 和 values，再利用 lambda x; x[0] 傳入 key 作為 sorted 排序的依據。

lambda 是不需要事先宣告的副程式，有點像 R 裡面寫 lapply (list, function (x) x^2)，將 function 嵌入程式碼當中。在排序的例子，items() 回傳 [key, value]，lambda x; x[0] 的意思是取出回傳項目索引值為 0 的東西。

R

雖然可以效法 python 的寫法，以 data.frame, list 或 vector 儲存計數，再使用迴圈計算各鹼基的數量。然而，R 內建許多方便的副程式可以統計向量內各項目的出現頻率。

簡言之，可利用 strsplit() 依照指定的分隔符（此例為 “”，即依照字符切割）將字串切割成子字串組成的向量，接著以 table() 統計向量內各項目的出現頻率。

bash script

bash script 要尋找字串也有許多寫法，其中一種做法是透過 grep -o 取得匹配的鹼基，再傳給 wc -l 計算列數（即字串中，指定鹼基符號的數量）。四種鹼基符號的統計值儲存於 bash array 並以空格間隔，以便輸出時辨識 array 各項目的數值。
